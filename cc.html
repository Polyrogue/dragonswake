CHARACTER CREATION
  1. All new characters must start out at level 1. Over levelled characters must be weakened/modified for entry into the Dragon’s Wake campaign. They will gain xp and items through the in-game world. All pre-existing inventories will not be available.
  2. Allowed races are:
    • Player's Handbook Races
    • Sword Coast Adventurer's Guide
    • Volo's Guide non-Monster races:
      • Aasimar
      • Firbolg
      • Goliath
      • Kenku
      • Lizardfolk
      • Tabaxi
      • Triton
  3. Allowed classes are:
    • Player's Handbook Classes
    • Sword Coast Adventurer's Guide Classes
    • The following Unearthed Arcana:
      • Revised Ranger
  4. Ability Scores are calculated using Point Buy.
    • We use the standard 5th Edition Point Buy Array (27 points).
    • Point Buy Calculator
  5. Character Sheets should be made using this character sheet template and submitted to a @DragonsWakeDM:
    • First, make a copy of the sheet in Google Drive
    • Once you’ve filled in your stats, set the sharing setting to “anyone can edit”
    • PM a share link to the document to any @DragonsWakeDM
  6. Hit points will be calculated by taking the average hp roll per level after 1st, as listed in the class features section of each class.
  7. Multiclassing is not allowed.
  8. The Pantheon used in this setting will be the Norse God Pantheon which can be found in the Player’s Handbook (PHB p299).
